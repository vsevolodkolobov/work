  • Что напечатает следующая программа?

    public class App {
      public static void main(String[] args) {
        int[] arr = {0, 1, 2, 3, 4};
        for (int k = 0; k < arr.length; k++) {
          arr[k] = arr[(k + 1) % arr.length];
        }
        System.out.println(java.util.Arrays.toString(arr));
      }
    }
