  • Что напечатает следующая программа?

    public class App {
      public static void main(String[] args) {
        labelA:
        for (int i = 0; i < 2; i++) {
          System.out.print("A");
          labelB:
          for (int j = 0; j < 2; j++) {
            System.out.print("B");
            if (i == j) {
              continue labelB;
            }
          }
        }
      }
    }
