    Объявление и инициализация переменной (A)

    Вопросы 10 , без ограничений по времени

    Начать тест

  • Скомпилируется ли данная программа?

    public class App {
      public static void main(String[] args) {
        x = 1;
      }
    }
    • да, скомпилируется

    • нет, программа содержит ошибки

      Используется необъявленная переменная 'x'

  • Скомпилируется ли данная программа?

    public class App {
      public static void main(String[] args) {
        int x;
      }
    }
    • да, скомпилируется

      Можно объявить, но не инициализировать переменную

    • нет, программа содержит ошибки

    • да, скомпилируется

    • нет, программа содержит ошибки
